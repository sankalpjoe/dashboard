# Quick Start Guide - New Features

## 🚀 What's New

Three major features have been added to your dashboard:

1. **AI Route Planner** - Smart routing with alternate routes
2. **OSINT Dorking** - Executive background checks via Google
3. **Sentiment Analysis** - Detect positive/negative news sentiment

Plus: **30-day filter** on Business Intel (no more old news!)

## 📍 How to Access

### Route Planner
1. Click **ROUTES** tab in navigation
2. Enter starting point (e.g., "New York")
3. Select from autocomplete dropdown
4. Enter destination (e.g., "Boston")
5. Click "Calculate Route"
6. View AI recommendations in "AI Alternates" tab

### OSINT Dorking
1. Click **INTELLIGENCE** tab in navigation
2. Select **OSINT Dorking** tab
3. Enter executive name (e.g., "Elon Musk")
4. Enter company name (e.g., "Tesla")
5. Click "Generate Dork Queries"
6. Copy queries or click search icon to open Google

### Sentiment Analysis
1. Click **INTELLIGENCE** tab in navigation
2. Select **Sentiment Analysis** tab
3. Paste any text (news, tweet, article)
4. Click "Analyze Sentiment"
5. View sentiment score, confidence, and breakdown

### Business Intel (Fixed)
1. Click **BUSINESS INTEL** tab
2. All news now filtered to last 30 days only
3. No more 1098-day-old articles!

- Type 3+ characters to see suggestions

### Copy Buttons
- Click copy icon to copy query
- Green checkmark confirms copied
- Paste directly into Google

### Sentiment Colors
- 🟢 Green = Positive
- 🔴 Red = Negative

## 🐛 Troubleshooting

### Routes not calculating?
- Make sure both origin and destination are selected
- Try typing full city name
- Check internet connection

### OSINT queries not working?
- Queries are generated correctly
- Click search icon to open Google
- Or copy and paste manually

### Sentiment seems wrong?
- Algorithm is keyword-based
- May miss sarcasm/context
- Best for straightforward news
