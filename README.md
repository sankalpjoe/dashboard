# DashINT: Strategic Intelligence & Situational Awareness

## Executive Summary
DashINT is a premium, real-time situational awareness platform designed for high-stakes intelligence monitoring. It integrates global news aggregation, multi-layered geospatial telemetry, and advanced AI-powered analytics into a unified, high-performance interface. Whether tracking maritime chokepoints, aviation surges, or regional geopolitical shifts, DashINT provides the "Strategic Posture" necessary for informed decision-making.

## Key Features

### 1. Unified Intelligence Surface
- **Live Intel Feed**: Real-time aggregation from 150+ curated RSS and OSINT sources.
- **Breaking News Ticker**: Instant alerts for critical global events with source-link integration.
- **Strategic Posture Assessment**: Composite risk scoring for regional stability and threat levels.

### 2. Advanced Geospatial Monitoring
- **Interactive 3D Map**: High-performance WebGL-based visualization with 40+ data layers.
- **Asset Tracking**: Real-time monitoring of personnel, tactical assets, and critical infrastructure.
- **Logistics Telemetry**: Integrated tracking for naval vessels (AIS) and military aircraft (ADS-B).
- **Infrastructure Overlays**: Mapping of undersea cables, pipelines, data centers, and military bases.

### 3. AI-Powered Analysis
- **Intelligence Briefs**: Automated, military-grade summaries for every major news event.
- **Regional Sentiment**: Analysis of linguistic and emotional trends across global news cycles.
- **Predictive Signals**: Identification of "Focal Points" where news, military, and infrastructure signals converge.

### 4. Enterprise-Grade Architecture
- **Multi-Platform**: Available as a high-performance Web App, PWA, and Native Desktop client (Tauri).
- **Security First**: Local-first AI support (Ollama) and encrypted credential management.
- **Offline Ready**: Cached map tiles and persistent data storage for field operations.

## How It Works
DashINT operates on a "Proto-first" architecture, using Protocol Buffers to define strict data contracts between client and server. The data pipeline fetches, cleans, and clusters signals every 60 seconds, ensuring the dashboard is always synchronized with the global environment.
